# Запуск бека:

## Установка всех нужных библиотек
```bash
python3 -m venv venv && pip3 install requirements.txt
```
## Запуск приложения

### Windows

```bash
source venv/bin/activate && python3 backend/app.py
````
### Linux
```bash
source venv/bin/activate && python3 backend/app.py
```

# Как узнать руоты?
http://127.0.0.1:8000/docs#/
